# E-Commerce-sales_Excel_Dashboard
Interactive Excel dashboard analyzing retail sales and profitability using the Sample Superstore Dataset

* **Data Cleaning & Integrity:** Cleaned and structured a 9,000+ row Sample Superstore Dataset, ensuring proper formatting for quantitative values and converting analytical labels like Postal Codes to Text.
* **Advanced Data Modeling:** Developed dynamic summaries utilizing core business metrics via advanced Excel functions like `=SUMIFS()` and `=AVERAGEIFS()`.
* **Business Insights Extraction:** Isolated a key profit leak in the **Furniture** category, discovering that aggressive promotional discounting on **Tables** generated a **-$17,725 net loss** despite high revenue.
* **Dashboard Design & Optimization:** Designed an interactive executive dashboard using **Line, Bar, and Pie Charts** coupled with **Dynamic Slicers** for instant regional filtering.
